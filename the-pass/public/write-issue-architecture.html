AI 編輯室怎麼寫一期

選題會拍板之後，/write-issue 怎麼把選出的稿，變成一期文章草稿。這是 The Pass 最核心的一塊。
選題決定寫什麼；/write-issue 負責怎麼寫。一個總指揮（orchestrator）調度四位 AI 編輯——各是獨立的工作分身——抓原文查證、各用自己的聲音寫、總編查核，中間設三個停點讓人拍板，最後產出草稿。核心精神：AI 起草、人定稿。

整條產線

能並行的並行、有依賴的排序；金色是「停下來給人決定」的關卡。

為什麼是「四個獨立分身」，不是一個 AI 全寫

如果讓同一個 AI 在同一個腦袋裡寫完三種編輯的稿，聲音會互相滲透——寫完 Mise 那種場景式的溫度，馬上接 Passe 的俐落，語氣會糊掉。所以每位編輯各開一個獨立分身（subagent），腦袋裡只裝自己的人格＋自己要寫的原文，其他什麼都不知道。聲音因此乾淨。總編也是獨立分身，沒參與寫作、純挑錯，不會自己審自己偏心。

角色寫什麼為什麼這樣切
orchestrator不寫稿；抓全文、調度、組裝、在 gate 問人、出稿全期的總指揮
Mise2 篇「今日觀察」長文（場景式、400–600 字）一個人寫兩篇 → 能差異化、不自我重複
Passe4 則「本期快訊」（事實先行、2–4 句）一起寫 → 不重疊、不撞長文角度
Fumet1 個結尾提問（從長文提煉、不選稿）讀完長文才提問，所以排在後面
總編7 項審核 + 逐條對事實獨立審 → 不偏心

三個停下來給人決定的關卡

GATE 1頭條長文寫完 → 停，確認聲音對不對、切角 OK 嗎，再寫其餘。
GATE 2總編查到數字／引述對不回原文 → 立刻停，問你：拿掉 / 去查證 / 你確認。不自己亂改。
GATE 3整期寫完、發佈前 → 停，你逐篇定奪：採用 / 要改哪 / 退。

三個守門原則

事實不可扭曲。引號＝原文真有人說過、數字＝原文。寫之前一定先抓全文。付費牆只有預覽 → 只報「現象」、透明標註、找公開來源佐證，不憑預覽編造。
AI 起草、人定稿。三個 gate 都停下來問人，AI 不替你做最終決定——跟選題的「AI 初選、人拍板」同一個哲學。
聲音隔離。四個分身各自獨立，聲音不互相污染、總編不自審。

怎麼讓編輯越寫越好

  • 調聲音規則（最快）：改寫作守則，下一期就生效。
  • 接編輯記憶（最深）：每期寫完把「寫了什麼、讀者反應」記起來，下期載入 → 三位編輯會記得自己寫過什麼、越來越貼品味。
  • 放好／壞對照範例：AI 學樣，比學抽象規則有效。
  • gate 的回饋就是訓練訊號：你在關卡改的東西（像把切角換掉），就是該回寫進守則的線索。

三層：為什麼 → 怎麼做 → 實際指令

每位編輯的「人格（Soul）」寫的是為什麼這樣寫（動機、價值觀、好惡，很少改）；「寫作守則（Style Guide）」是怎麼做（可執行的規則）；跑的時候把兩者＋原文＋（未來的）記憶組起來，才成為那位編輯實際的指令。改人格影響的是「她在乎什麼、選什麼角度」，比改規則更深層。
