電子報編輯室怎麼寫一期

選題會拍板之後，/write-issue 怎麼把選出的稿，變成一期文章草稿。這是 The Pass 最核心的一塊。
選題決定寫什麼；/write-issue 負責怎麼寫。一個總指揮（orchestrator）調度四位 AI 編輯——各是獨立的工作分身——抓原文查證、各用自己的筆調寫、總編查核，中間設三個停點讓人拍板，最後產出草稿。核心精神：AI 起草、人定稿。

整條產線

能並行的並行、有依賴的排序；金色是「停下來給人決定」的關卡。

為什麼是「四個獨立分身」，不是一個 AI 全寫

如果讓同一個 AI 在同一個腦袋裡寫完三種編輯的稿，筆調會互相滲透——寫完 Mise 那種場景式的溫度，馬上接 Passe 的俐落，語氣會糊掉。所以每位編輯各開一個獨立分身（subagent），腦袋裡只裝自己的人格＋自己要寫的原文，其他什麼都不知道。筆調因此乾淨。總編也是獨立分身，沒參與寫作、純挑錯，不會自己審自己偏心。

角色寫什麼為什麼這樣切
orchestrator不寫稿；抓全文、調度、組裝、在 gate 問人、出稿全期的總指揮
Mise2 篇「今日觀察」長文（場景式、400–600 字）一個人寫兩篇 → 能差異化、不自我重複
Passe4 則「本期快訊」（事實先行、2–4 句）一起寫 → 不重疊、不撞長文角度
Fumet1 個結尾提問（從長文提煉、不選稿）讀完長文才提問，所以排在後面
總編7 項審核 + 逐條對事實獨立審 → 不偏心

三個停下來給人決定的關卡

GATE 1頭條長文寫完 → 停，確認筆調對不對、切角 OK 嗎，再寫其餘。
GATE 2總編查到數字／引述對不回原文 → 立刻停，問你：拿掉 / 去查證 / 你確認。不自己亂改。
GATE 3整期寫完、發佈前 → 停，你逐篇定奪：採用 / 要改哪 / 退。

三個守門原則

事實不可扭曲。引號＝原文真有人說過、數字＝原文。寫之前一定先抓全文。付費牆只有預覽 → 只報「現象」、透明標註、找公開來源佐證，不憑預覽編造。
AI 起草、人定稿。三個 gate 都停下來問人，AI 不替你做最終決定——跟選題的「AI 初選、人拍板」同一個哲學。
筆調隔離。四個分身各自獨立，筆調不互相污染、總編不自審。

怎麼讓編輯越寫越好

  • 調筆調規則（最快）：改寫作守則，下一期就生效。
  • 接編輯記憶（最深，已接 2026-06）：每期寫完把「學到的準則、寫過的主題」記起來，下期載入 → 三位編輯記得自己學過／寫過什麼、越來越貼品味。詳見下方〈編輯記憶〉。
  • 放好／壞對照範例：AI 學樣，比學抽象規則有效。
  • gate 的回饋就是訓練訊號：你在關卡改的東西（像把切角換掉），就是該回寫進守則的線索。

編輯記憶：怎麼讓三位編輯越寫越像「自己」

這是 The Pass 養「虛擬 KOL」的核心——校正過的東西不會說完就忘，會永久變成編輯的能力。（2026-06 接上）

每位編輯有三層設定，跑稿時組在一起，才成為他實際的「大腦」：

層是什麼（一句話）多久變
Soul · 人格我是誰——價值觀、背景、好惡（＝為什麼這樣寫）很少變
Memory · 記憶我學到什麼、寫過什麼（隨時間長）每期長
Voices · 守則怎麼寫——可執行的規則與招式偶爾調

其中「記憶」放兩種東西：

① 我學到的準則。總編／Terrel 校正過、會「改變以後每期怎麼寫」的規則，用第一人稱記成編輯自己的學習筆記。例：首期標題被改過後，Mise 記下「標題要看得出主題＋勾得起好奇，兩端都不能偏」——下期它一動筆就記得。
② 我寫過的主題。每期寫了哪些題目／標題各記一行，下期載入 → 編輯記得自己寫過什麼、不重複角度。滾動保留近 ~12 期，不讓檔變肥。

飛輪怎麼轉（每期一圈）：

① 開寫前
每位編輯載入自己的 Soul＋Memory＋Voices
→
② 寫稿
Mise 動筆前已讀到「上次標題被改、要看得出主題」
→
③ 定稿後
回寫記憶：主題自動記、準則你確認才記
→
④ 下一期
帶著上期的學習寫 → 越來越像那個人
↻ 每期跑一圈，三位編輯跨期累積、自然演化，不用每次重講
誰決定記什麼（品質的關，人把）。「寫過的主題」這種事實，每期自動記；像標題鐵則這種準則級的，AI 先提議、Terrel 點頭才寫進記憶——跟選題、寫稿同一個「AI 起草、人定稿」哲學。

這個 skill 由什麼組成（檔案架構）
