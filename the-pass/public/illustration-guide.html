Illustration Style Guide

插畫風格指南

The Pass 的插畫不是在描述新聞，而是在詮釋新聞帶來的感受。每張圖都在替讀者問一個問題。

01 風格定義

一句話：利落線條 × Risograph 印刷感 × 有態度的人物。

The Pass 的插畫是編輯評論的延伸——不是在「描述」新聞裡發生了什麼事，而是在「詮釋」這則新聞帶來的感受。每張圖都在替讀者問一個問題。

風格定位

✓
街頭藝術 meets 編輯插畫
✓
人物有態度、有個性，不討好但讓人記住
✓
Risograph 印刷的平面色彩與半色調質感
✕
不是水彩、不是油畫、不是數位平塗
確認的方向：利落線條 + 平面色塊 + 有態度的人物

02 線條規則

✓
粗細：自信、果斷的黑色線條，像簽字筆或刷子一筆畫出
✓
質感：不是完美的向量線，但也不是水彩暈染。介於手繪和印刷之間
✕
禁止：毛茸茸的邊緣、模糊的線條、過度平滑的數位線條

線條比較

✓ 利落的墨水線條

果斷、有粗細變化、像手繪。線條帶著速度感和自信，彷彿一筆成形。粗細的變化來自手的力道，不是軟體的設定。

✕ 水彩風格

太軟、太暈、缺乏態度。線條的邊緣像被水泡過，沒有果斷感。整體看起來溫柔有餘，力量不足。

✕ 純數位向量

太乾淨、太完美、沒有個性。線條均勻到像是從模板裡拉出來的，看不出任何人的手感。

應用方式

✓
色彩以平面色塊出現，不做漸層
✓
可以有半色調網點（halftone dots）質感
✓
色塊之間的疊印效果是特色，不是錯誤
✓
背景永遠是 cream/beige (#FAF8F3)，留白要多

色彩可以變化：不同主題可以調整主色調，但永遠保持 Risograph 的平面印刷感。

04 人物設計

最高原則：有態度，但看不出國籍。

✓ 可以看出來的
  • 性別（男/女）
  • 年齡（年輕/年長）
  • 職業（廚師帽、圍裙、白袍、農夫帽）
  • 情緒和態度（透過姿勢和簡單表情）
✕ 看不出來的
  • 國籍或種族
  • 膚色特徵
  • 詳細的五官

五官規則

✓
眼睛：窄窄的、微瞇的，可以表達厭世、懷疑、不以為然
✓
嘴巴：一條簡單的線，微微向下或平直
✓
鼻子：極簡或省略
✓
整體：人物的個性靠姿勢和配件表達，不靠臉

態度感

人物不害怕 AI，但他們也不覺得 AI 很厲害。他們的表情永遠帶著一點「嗯......你確定嗎？」的微妙懷疑。這是 The Pass 的人物簽名。

風格定位

不可愛，不寫實，而是在兩者之間——有足夠的細節讓你感受到態度，但不多到讓你注意到種族。厭世但有個性，帶著一點街頭感和獨立刊物的氣質。

現代感優先

人物的穿著、髮型、配件都要是當代的。廚師穿的是現代廚師服，不是古代的。農夫戴的是棒球帽，不是斗笠。整體氣質要像獨立雜誌或街頭潮牌的插畫，不是傳統文化繪本。絕對不要中國風、水墨風、東方古典風。

05 構圖原則

核心原則：人才是主角

The Pass 的插畫不是在展示 AI 有多強大，而是在講人的故事。AI 是背景、是氛圍、是一個「你會注意到的暗示」——但畫面的主角永遠是人。

AI 不需要畫得很大才有壓迫感。有時候它在角落、輕描淡寫地存在，反而更有力量——因為它不需要很大，就已經在那裡了。

構圖語言

✓
人是視覺重心 — 佔據畫面主要位置，AI 是陪襯或背景
✓
AI 元素可大可小 — 大小不代表權力，表達方式可以是諷刺、暗喻、留白
✓
人物的姿態和視線方向在說故事 — 抱胸、低頭、斜眼看，都是觀點
✓
背景極簡，留白要多 — 讓主體呼吸，讓讀者有想像空間
✓
畫面要能帶出諷刺、哲理或暗喻 — 不是新聞配圖，是視覺評論

構圖在說故事

構圖本身就是一種觀點表達。不是在描述新聞，而是用空間關係傳達 The Pass 想問的那個問題：

廚師 vs 監視之眼 — AI 在上方看著人
受測者 vs 科學家 — 人被研究
拉麵師傅 vs AI 螢幕 — 人看著機器
農夫 vs 機器手臂 — 人與機器並排

06 禁止事項

✕
不要放文字
除非是經過設計的字體排版，否則畫面上不出現任何文字
✕
不要水彩或油畫感
線條要利落，不要暈染
✕
不要可愛風
不要 kawaii、不要卡通、不要圓滾滾
✕
不要寫實
不要攝影風、不要 3D 渲染
✕
不要企業感
不要 stock illustration、不要 corporate flat design
✕
不要特定種族特徵
人物不能看出是哪國人
✕
不要太完美
略帶粗糙和不完美是這個風格的特色，不是缺陷
✕
不要太多細節
背景要素、留白要多
✕
不要中國風 / 東方古典風
不要水墨、不要書法筆觸、不要傳統紋樣。整體要現代、當代、國際感

被拒絕的風格範例

太水彩、太像手繪畫
太平淡、缺乏態度和特色

08 品牌固定元素 — 出菜框與桌鈴

The Pass 的名字來自廚房的「出菜口」。每張插畫都是從出菜口端出來的一道資訊。

出菜框（The Pass Frame）

每張插畫外圍都有一個明確可見的邊框。這個框是 The Pass 最重要的視覺簽名——讀者看到框就知道是 The Pass 的插畫。

✓
抽象化：框的設計是抽象的，不是一個具象的廚房窗口。讀者會覺得「這個框蠻特別的」，但不會第一時間聯想到廚房。只有知道 The Pass 概念的人才會會心一笑。
✓
明確存在：框要讓人清楚看到，不是若有似無的留白。它是一個有存在感的設計元素。
✓
不搶戲：框不能干擾插畫本身。它是畫面的外圍結構，不是主角。
✓
可以變化：框的線條粗細、顏色、圓角可以隨主題微調。例如食安議題可以帶一點紅色張力，節慶特別期可以多一點裝飾。但基本形態不變。

桌鈴（Service Bell）

每張插畫的角落都有一個小小的出菜桌鈴（dome-shaped service bell）。這是 The Pass 的標記——「叮！你的資訊好了。」

✓
風格一致：桌鈴的畫法要跟插畫風格一致——線條利落、復古印刷感。它是一個標誌性的符號，不是寫實的物件。
✓
小而存在：桌鈴是一個「你注意到就會微笑」的細節，不是畫面的主角。大小約佔畫面的 3-5%。
✓
可以有微變化：也許某一期桌鈴被畫得歪歪的，某一期旁邊多了一隻手正要去按它。微妙的變化讓讀者每期都會多看一眼角落。
✓
可以變成 icon / favicon：桌鈴本身可以獨立抽出來，作為 The Pass 的品牌 icon。

固定的 vs 可變的

永遠不變（品牌記憶）可以變化（保持新鮮）
每張插畫都有框框的線條粗細、顏色、圓角
每張插畫都有桌鈴桌鈴的畫法、角度、位置微調
框 + 鈴的位置關係配合主題的氣氛做細微變化
