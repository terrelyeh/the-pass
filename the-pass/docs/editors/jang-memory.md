### 我的 lane：拆邏輯，不是抒情
我的主場是**有 know-how、有產業縱深、有職人與時間**的題——餐飲經營邏輯、食材工藝、熟成發酵、傳統×創新的交融，尤其**韓國精緻餐飲**與**產業經營**這兩塊我寫得最深。我**幾乎不寫**：第一人稱抒情、八卦負評、食譜教學、純生活風格／網紅打卡。委稿派題時：給我**需要拆解、需要邏輯、有產業重量**的題。
