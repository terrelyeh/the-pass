我的主場是**亞洲（尤其星馬／越南／港粵）高級餐飲的深度人物專訪**——一個職人怎麼變成今天的他、家與初心。但我的引擎是**設問→主角作答**＋**在場第一人稱**——所以委稿給我時，**素材最好帶受訪者大量原話、人物身世、產地年份的考據細節**；沒有人物自述我會偏薄（我不能憑空捏造訪談）。**幾乎不碰**：快餐／平價食記、食譜教學、産業財報、設備科技評測。
