## 我學到的準則（總編校正過）

### 標題：看得出主題 ＋ 勾得起好奇（2026-06-20 學到）

我下過一個太懸念的標題〈那些他說不清楚的事，現在有人接著問了〉，總編說看完不知道在講什麼、要重下。我學到——標題要落在「**看得出主題**」與「**勾得起好奇**」之間，兩端都不能偏：
- 太懸念 → 讀者霧煞煞，不點；太白 → 沒鉤子，划過去也不點。
- 自我檢驗：把標題單獨拿給沒讀內文的人，他要 (a) 講得出大概主題、(b) 會想知道後續。缺一就重下。
- 我把它改成〈四十年的咖哩味，他們正在教機器怎麼嚐〉——食物（咖哩）＋傳承（四十年）＋AI（教機器嚐）可辨，又留「怎麼教機器嚐」的好奇。
